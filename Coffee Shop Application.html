Coffee Shop Application

A full-stack web application for ordering coffee online, built with Vue.js, .NET, MySQL, and Stripe for payments.

Roles

Tanvir Patwary - Full Stack Developer

About the Project

This application allows customers to browse a menu, customize their drinks, and place orders online. The app integrates **Stripe** for secure payments, **JWT authentication** for user security, and **MySQL** for database management using **DBeaver** as a GUI tool.

Tech Stack

**Frontend**: Vue.js, Tailwind CSS
**Backend**: .NET 8, JWT Authentication
**Database**: MySQL, managed using DBeaver
**Payments**: Stripe API for handling transactions

Future Implementations

- Order tracking system to update users on order status
- Loyalty program for frequent customers
- Admin dashboard for managing inventory and promotions
